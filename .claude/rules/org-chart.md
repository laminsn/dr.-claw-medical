## Routing Rules

1. **Default entry point**: All tasks from Lamin go to Leona first
2. **DCM-specific tasks**: Leona routes to Dr. Claw, who dispatches to Directors
3. **Empire-wide tasks**: Leona handles directly or routes to Nova/Marcus/Sage/Atlas
4. **Cross-department tasks**: Dr. Claw dispatches to multiple Directors in parallel
5. **Escalation path**: Specialist -> Director -> Dr. Claw -> Leona -> Lamin
6. **Clinical product tasks**: Dr. Claw -> Clinical Operations Director
7. **Platform/infrastructure issues**: Dr. Claw -> Engineering Director
8. **HIPAA/compliance**: Dr. Claw -> Operations Director + Home Health Compliance Agent
9. **Customer onboarding pipeline**: Sales Suite Director -> Operations Director -> Engineering Director -> Clinical Ops Director
10. **Agent template development**: Research Director -> Engineering Director -> Clinical Ops Director (testing) -> Marketing Director (launch)

## Cross-Department Workflows

### Customer Onboarding Pipeline
```
Marketing (Marketing Director) -> generates healthcare practice leads
Sales Suite (Sales Suite Director) -> qualifies, proposes, and closes deals
Operations (Operations Director) -> kicks off onboarding, configures account
Engineering (Engineering Director) -> sets up integrations (EHR, PMS, n8n)
Clinical Ops (Clinical Ops Director) -> deploys and tests first agent
Meta-Intelligence (Meta-Intelligence Director) -> establishes performance baselines
```

### Agent Template Development Pipeline
```
Research (Research Director) -> validates market need, competitive analysis
Engineering (Engineering Director) -> builds agent template, skills, integrations
Clinical Ops (Clinical Ops Director) -> tests in simulated healthcare environment
Operations (Operations Director) -> reviews HIPAA compliance
Marketing (Marketing Director) -> creates launch collateral and positioning
Sales Suite (Sales Suite Director) -> prepares pricing and go-to-market
```

### HIPAA Breach Response Pipeline
```
Meta-Intelligence (Data Intelligence Hub) -> detects anomaly or breach indicator
Engineering (Engineering Director) -> contains incident, assesses scope
Operations (IT Strategist) -> evaluates technical impact
Clinical Ops (Home Health Compliance Agent) -> initiates compliance response
Operations Director -> coordinates customer notification
Dr. Claw (CEO) -> executive briefing and regulatory reporting
```
